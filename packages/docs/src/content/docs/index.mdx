## Workflow Overview

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│  User Request   │────▶│ Producer Skills │────▶│   .claude/.specs/│
└─────────────────┘     │  - brainstorming│     │   (spec files)   │
                        │  - debugging    │     └────────┬─────────┘
                        │  - refactoring  │              │
                        │  - reviewing    │              ▼
                        └─────────────────┘     ┌─────────────────┐
                                                │ Consumer Skill  │
                                                │  /implementing  │
                                                └────────┬────────┘
                                                         │
                                                         ▼
                                                ┌─────────────────┐
                                                │ Code Output     │
                                                └─────────────────┘
```

## Skills Categories

### Producer Skills (4)

Tạo specification files từ yêu cầu của user:

| Skill           | Mục đích                           |
| --------------- | ---------------------------------- |
| `/brainstorming` | Ideation, feature design, architecture |
| `/debugging`    | Bug analysis, root cause analysis  |
| `/refactoring`  | Code smell detection, migration planning |
| `/reviewing`    | Code review, security audit        |

### Consumer Skill (1)

| Skill           | Mục đích                           |
| --------------- | ---------------------------------- |
| `/implementing` | Execute specifications into code   |

### Utility Skills (3)

| Skill                   | Mục đích                           |
| ----------------------- | ---------------------------------- |
| `/ui-ux-pro-max`        | UI/UX design guidelines            |
| `/mermaidjs-v11`        | Create diagrams                    |
| `/react-best-practices` | React/Next.js optimization         |

## Quick Installation

```bash
# Clone repository
git clone https://github.com/ltvai/ltvai-kit.git

# Copy .claude folder to your project
cp -r ltvai-kit/.claude your-project/
```

Xem hướng dẫn chi tiết tại [Installation Guide](/getting-started/installation/).
